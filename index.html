Objetivo del laboratorio

Crear páginas web básicas usando HTML y CSS. Aprender y practicar las etiquetas HTML básicas (texto, listas, enlaces, tablas) y aplicar estilos con CSS.

Prerrequisitos:

  • Tener una cuenta de GitHub
  • Instalar Visual Studio Code
  • Navegador Web

¿Para qué sirven las siguientes etiquetas?

<!DOCTYPE html>
Indica que el documento utiliza HTML5 y obliga al navegador a renderizar en modo estándar.
<html>
Elemento raíz del documento. El atributo lang="es" indica el idioma principal.
<head>
Contiene metadatos (por ejemplo: <meta>, <title>), enlaces a CSS y scripts.
<meta>
Proporciona metadatos como charset (juego de caracteres) o viewport (para diseño responsive).
<title>
Título de la página que aparece en la pestaña del navegador y en resultados de búsqueda.
<body>
Contenido visible de la página: texto, imágenes, tablas, formularios, etc.
